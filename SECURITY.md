# Security

本项目仅展示个人 AI 学习工作流，不应包含任何敏感信息。

## 不应提交

- DeepSeek / Tavily / 百度地图 / OpenRouter / Notion 等 API Key
- Claude / ChatGPT / Codex 账号信息
- 私人手机号、邮箱、身份证件
- 密钥截图
- 带有真实 token 的配置文件
- 本地系统敏感路径截图

## 安全原则

- Skills 可以公开，但 API Key 不可以公开
- MCP URL 中如果包含 Key，必须打码或移除
- 高权限工具应限制在 AI-Sandbox 工作区
- 不建议长期开放任意网址访问权限
